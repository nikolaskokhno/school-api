## Routes

| Method  | Route              | Description      |
| :-------|:------------------:| ----------------:|
| GET     | /{entityName}/     | returns all item |
| POST    | /{entityName}/     | add item         |
| DELETE  | /{entityName}/:id  | delete item      |
| PUT     | /{entityName}/:id  | update item      |
| GET     | /{entityName}/:id  | return one item  |
